This professional chatbot features:

  • Modern, clean UI with smooth animations
  • Real-time REST communication
  • Broadcast notifications when chat is closed
  • Connection status indicators
  • Message history with timestamps
  • Full-screen support for immersive conversations
  • Support for Markdown and image attachments
  • Responsive design for all devices

The chatbot UI supports broadcast messages. To trigger a broadcast, your backend must send a POST request to https://your-rest-api-endpoint/broadcast. The /broadcast should receive a JSON payload like: 

{
  "text": "Hello, Qsc team!"
}

<qsc-chatbot
